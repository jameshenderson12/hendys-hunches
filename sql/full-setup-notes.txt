Setup Steps
===========

1. Ensure all database tables contain 'live' prefix.
2. Ensure database settings are correct in 'db-connect.php'.
3. Update all with 'live' db prefix. Do a find/replace all in root folder.
4. Update number of score_p and score_r values 'process.php'.




Password required elements:
(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{6,}
