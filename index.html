Hendy's Hunches: How It Works

To play Hendy's Hunches, you will soon need to:

  1. Register to play (registration link available soon!)
  2. Login using your registered username and password
  3. Start recording/updating your predictions on the fixtures form
  4. Check the 'Rankings' page after each match to see where you (and your colleagues, friends or family) stand!

For any match prediction, you can be awarded either 0, 1, 2, 3 or 7 points.

The different scenarios for points scoring are as follows:

  1. 1 point is awarded if you correctly predict either the home or away score (goals).
  2. 2 points are awarded if you correctly predict any match outcome of home win, away win or draw.
  3. 3 points are awarded if you correctly predict the match outcome and either the home or away score (scenario A + B).
  4. 7 points are awarded if you correctly predict both home and away scores.
